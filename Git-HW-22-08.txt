

1. Как отказаться от изменения в файле (файлах), находящегося в рабочем каталоге (work directory)?
Если какой-то файл был изменен и необходимо отменить изменения в отдельном файле, используйте команду git checkout <имя_файла>. Чтобы внести изменения во все файлы, используйте git checkout . команда.
.
2. Как вернуть файл из индекса в рабочий каталог?
Если какой-то файл был загружен, и он должен вернуть файл из индекса в рабочий каталог, используйте команду git restore --staged <имя_файла>.
.
3. Как отменить ошибочный commit ?
Если какой-то файл был зафиксирован и в него необходимо внести изменения, используйте команду git commit --amend, измените файл и зафиксируйте его снова. Чтобы отменить фиксацию, помогает команда git reset HEAD~<количество отмененных коммитов>. “
.
